Dictionary Workbench Ondict

Look up words, or search by meaning.

Headword search jumps straight to the entry. Definition search scans the SQLite full-text index and returns candidate words with context snippets.

    Advanced options

    Autocomplete stays on for headword mode and turns off for definition mode.

    What changed

    The HTML server now has a consistent shell across home, results, and entry pages, so you can browse like a small web app instead of jumping between unrelated pages.

    Tips

    • Use headword mode for fast dictionary lookup.
    • Use definition mode when you know the concept but not the word.
    • If definition search says the index is missing, rebuild `vocab.db` with the dump/init flow.
